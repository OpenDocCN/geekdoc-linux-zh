# Modify the HTTP request

As described earlier, each HTTP transfer starts with curl sending an HTTP request. That request consists of a request line and a number of request headers, and this chapter details how you can modify all of those.

*   [Request method](ch170.xhtml#http__modify__method__md)
*   [Request target](ch171.xhtml#http__modify__target__md)
*   [Fragment](ch172.xhtml#http__modify__fragment__md)
*   [Customize headers](ch173.xhtml#http__modify__headers__md)
*   [Referer](ch174.xhtml#http__modify__referer__md)
*   [User-agent](ch175.xhtml#http__modify__user-agent__md)

Of course, changing the [HTTP version](../versions/) is another way to alter the request.
