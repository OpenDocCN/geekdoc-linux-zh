# 关于本书

+   **本版本于 2024 年 8 月 2 日发布**

这是一本开源的 Bash 脚本入门指南，它将帮助你学习 Bash 脚本的基础知识，并开始编写出色的 Bash 脚本，这些脚本将帮助你自动化日常的 SysOps、DevOps 和 Dev 任务。无论你是 DevOps/SysOps 工程师、开发者，还是仅仅是一个 Linux 爱好者，你都可以使用 Bash 脚本来组合不同的 Linux 命令，自动化繁琐和重复的日常任务，这样你就可以专注于更有生产力和更有趣的事情。

该指南适合任何作为开发者、系统管理员或 DevOps 工程师工作，并希望学习 Bash 脚本基础知识的人。

前面的 13 章将完全专注于构建一些扎实的 Bash 脚本基础，其余章节将提供一些实际案例和脚本。
