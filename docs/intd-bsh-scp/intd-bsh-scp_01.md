# Bash 脚本入门

欢迎来到这个 Bash 基础培训指南！在这个 **bash 快速入门课程** 中，你将学习 **Bash 基础**，这样你就可以开始编写自己的 Bash 脚本并自动化你的日常任务。

Bash 是一个 Unix shell 和命令语言。它在各种操作系统上广泛可用，也是大多数 Linux 系统上的默认命令解释器。

Bash 代表 Bourne-Again SHell。与其他 shell 一样，你可以在终端中直接以交互方式使用 Bash，也可以像使用任何其他编程语言一样使用 Bash 来编写脚本。这本书将帮助你学习 Bash 脚本的基础知识，包括 Bash 变量、用户输入、注释、参数、数组、条件表达式、条件语句、循环、函数、调试和测试。

Bash 脚本非常适合自动化重复性工作负载，并且能帮你节省大量时间。例如，想象一下你和五名开发者一起在一个需要繁琐环境设置的项目上工作。为了让程序正确运行，每个开发者都必须手动设置环境。这就是同一个非常长的任务（设置环境）至少重复五次。这就是你和 Bash 脚本发挥作用的地方！所以，你创建一个包含所有必要指令的简单文本文件，并与你的队友分享。现在，他们只需要执行 Bash 脚本，所有事情都会为他们自动完成。

为了编写 Bash 脚本，你只需要一个 UNIX 终端和一个文本编辑器，如 Sublime Text、VS Code，或者基于终端的编辑器，如 vim 或 nano。
