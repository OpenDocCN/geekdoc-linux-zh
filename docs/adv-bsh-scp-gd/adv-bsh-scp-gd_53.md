这是对 **sed** 和 **awk** 文本处理工具的非常简短的介绍。在这里，我们只处理几个基本的命令，但这足以理解在 shell 脚本中简单的 sed 和 awk 构造。

**sed**：一个非交互式的文本文件编辑器

**awk**：一种以 **C** 风格语法的面向字段的模式处理语言

尽管这两个工具存在许多差异，但它们具有相似的调用语法，使用 正则表达式 ，默认从 `stdin` 读取输入，并将输出到 `stdout`。这些是行为良好的 UNIX 工具，并且它们可以很好地协同工作。一个工具的输出可以被管道传输到另一个工具，它们结合的能力赋予了 shell 脚本一些 Perl 的功能。
