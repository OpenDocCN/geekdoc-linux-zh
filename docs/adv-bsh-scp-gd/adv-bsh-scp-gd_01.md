Shell 是一个命令解释器。它不仅是操作系统内核与用户之间的隔离层，而且也是一种相当强大的编程语言。所谓的 Shell 程序，即*脚本*，是一种通过“粘合”系统调用、工具、实用程序和编译后的二进制文件来构建应用的易于使用的工具。几乎所有的 UNIX 命令、实用程序和工具都可以通过 Shell 脚本调用。如果还不够，内部 Shell 命令，如测试和循环结构，也为脚本提供了额外的功能和灵活性。Shell 脚本特别适合于系统管理任务和其他不需要完整结构化编程语言花哨功能的常规重复性任务。
