# 38.2\. 关于作者

> 原文：[`tldp.org/LDP/abs/html/aboutauthor.html`](https://tldp.org/LDP/abs/html/aboutauthor.html)

这个人究竟是谁呢？

作者声称没有资历或特殊资格，[[1]](#FTN.AEN21292) 除了写作的强迫症。[[2]](#FTN.AEN21295)

这本书在一定程度上偏离了他其他的主要作品，《如何遇见女人：害羞男人的关系指南》(http://bash.deta.in/hmw60.zip)。他还写了《软件构建指南》(http://tldp.org/HOWTO/Software-Building-HOWTO.html)。最近，他尝试了他的（重）手笔写小说：《柏林上空的戴夫·道森（第一部分）》(http://bash.deta.in/dave-dawson-over-berlin.epub)、《柏林上空的戴夫·道森（第二部分）》(http://bash.deta.in/dave-dawson-over-berlin.II.epub) 和《柏林上空的戴夫·道森（第三部分）》(http://bash.deta.in/dave-dawson-over-berlin.III.epub)。他还拥有一些*Instructables*([这里](http://www.instructables.com/id/Arduino-Morse-Code-Shield/), [这里](http://www.instructables.com/id/Haywired-Hackduino/), [这里](http://www.instructables.com/id/Arduino-DIY-SD-Card-Logging-Shield/), [这里](http://www.instructables.com/id/Binguino-An-Arduino-based-Bingo-Number-Generato/), [这里](http://www.instructables.com/id/The-Raspberry-Pi-Lapdock-Connection/), [这里](http://www.instructables.com/id/The-Raspberry-Pi-Arduino-Connection/), 和[这里](http://www.instructables.com/id/Switchable-Dual-Voltage-33v5v-Hacduino/))，这些是他的（不）荣誉。

自 1995 年以来一直是 Linux 用户（Slackware 2.2，内核 1.2.1），作者发布了一些软件，包括一次性密码加密工具[cruft](http://ibiblio.org/pub/Linux/utils/file/cruft-0.2.tar.gz)，按揭计算器[mcalc](http://ibiblio.org/pub/Linux/apps/financial/mcalc-1.6.tar.gz)，Scrabble 裁判[judge](http://ibiblio.org/pub/Linux/games/amusements/judge-1.0.tar.gz)，单词游戏列表包[yawl](http://ibiblio.org/pub/Linux/libs/yawl-0.3.2.tar.gz)，以及 Quacky 字谜游戏包[Quacky](http://bash.deta.in/qky.README.html)。他在计算机游戏方面起步相当艰难——在 CDC 3800 上编程 FORTRAN IV（在纸上编码垫上，偶尔使用键盘穿孔机和 Friden Flexowriter），并且对那些日子并不怀念。

与妻子和橘色虎斑猫居住在一个偏远社区，他珍视人类的脆弱性，尤其是自己的。[[3]](#FTN.AEN21318)
