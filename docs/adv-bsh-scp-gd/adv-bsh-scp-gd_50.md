# 参考文献列表

> 原文：[`tldp.org/LDP/abs/html/biblio.html`](https://tldp.org/LDP/abs/html/biblio.html)

|  **那些不了解 UNIX 的人注定要拙劣地重新发明它。**

*--亨利·斯宾塞**  |

由彼得·登宁编辑，*《计算机遭受攻击：入侵者、蠕虫和病毒》*，ACM 出版社，1990 年，0-201-53067-8。

这本汇编包含了关于 shell 脚本病毒的一些文章。

*

肯·伯特奇，*[使用 Bash 进行 Linux Shell 脚本编程](http://www.samspublishing.com/title/0672326426)*，第 1 版，Sams 出版社（培生），2004 年，0672326426。

覆盖了与 *ABS Guide* 相同的大量材料，但风格不同。

*

丹尼尔·戈德曼，*[Sed 的权威指南](http://www.sed-book.com/)*，第 1 版，2013 年。

这本电子书是 *sed* 的优秀入门介绍。它不是从印刷版转换而来，而是专门设计和格式化，以便在电子书阅读器上查看。内容编写得很好，信息丰富，既可作为参考，也可作为教程。强烈推荐。

*

戴尔·多赫蒂和阿诺德·罗宾斯，*Sed 和 Awk*，第 2 版，O'Reilly and Associates，1997 年，1-156592-225-5。

要完全发挥 shell 脚本的力量，至少需要对 *sed* 和 *awk* 有一定的了解。这是经典的教程，包括对 *正则表达式* 的优秀介绍。推荐阅读。

*

杰弗里·弗里德兰，*精通正则表达式*，O'Reilly and Associates，2002 年，0-596-00289-0。

仍然是关于 正则表达式 的最佳全面参考。

*

Aeleen Frisch，*系统管理精华*，第 3 版，O'Reilly and Associates，2002 年，0-596-00343-9。

这本优秀的手册从系统管理员的角度提供了对 shell 脚本的 decent 介绍。它包括对 UNIX 系统中启动和初始化脚本的全面解释。

*

斯蒂芬·科 chan 和帕特里克·伍德，*Unix Shell 编程*，海登出版社，1990 年，067248448X。

尽管有些过时，但仍然被视为标准参考书籍，从风格上讲有点“生硬”。[[1]](#FTN.AEN21579) 事实上，这本书是 *ABS Guide* 作者多年前第一次接触 UNIX shell 脚本。

*

尼尔·马修和理查德·斯托尼斯，*Linux 编程入门*，Wrox 出版社，1996 年，1874416680。

意想不到地对 Linux 可用的各种编程语言进行了深入覆盖，包括一个相当强的 shell 脚本章节。

*

赫伯特·梅耶，*在 IBM PC 上的高级 C 编程*，Windcrest 出版社，1989 年，0830693637。

对算法和一般编程实践有很好的覆盖。强烈推荐，但遗憾的是已绝版。

*

大卫·梅迪内茨，*Unix Shell 编程工具*，麦格劳-希尔出版社，1999 年，0070397333。

对 shell 脚本有很好的处理，包括示例，并对 Tcl 和 Perl 进行了简短的介绍。

*

卡梅隆·纽汉姆和比尔·罗森布拉特，*学习 Bash Shell*，第 2 版，O'Reilly and Associates，1998 年，1-56592-347-2。

这是对一个不错的 shell 入门教程的勇敢尝试，但遗憾的是在编写脚本方面的覆盖不足，并且例子也不够充分。

*

Anatole Olczak，*Bourne Shell 快速参考指南*，ASP，Inc.，1991，093573922X。

尽管缺乏对 Bash 特定功能的覆盖，但这是一本非常实用的口袋参考。

*

Jerry Peek，Tim O'Reilly 和 Mike Loukides，*Unix 实用工具*，第 3 版，O'Reilly and Associates，Random House，2002，0-596-00330-7。

包含几个关于 shell 编程的非常详尽和深入的文章部分，但不足以成为自学手册。它复制了上面 Dougherty 和 Robbins 书中的大部分正则表达式教程。对 UNIX 命令的全面覆盖使这本书值得放在你的书架上。

*

Clifford Pickover，*计算机、模式、混沌与美*，St. Martin's Press，1990，0-312-04123-3。

一座充满想法和计算机探索数学奇点的食谱宝库。

*

George Polya，*如何解决问题*，普林斯顿大学出版社，1973，0-691-02356-5。

这是对问题解决方法（算法）的经典教程，特别强调如何教授它们。

*

Chet Ramey 和 Brian Fox，*[GNU Bash 参考手册](http://www.network-theory.co.uk/bash/manual/)*，Network Theory Ltd，2003，0-9541617-7-7。

这本手册是 GNU Bash 的权威参考。本手册的作者 Chet Ramey 和 Brian Fox 是 GNU Bash 的原始开发者。每售出一本，出版商向自由软件基金会捐赠 1 美元。

*

Arnold Robbins，*Bash 参考卡*，SSC，1998，1-58731-010-5。

优秀的 Bash 口袋参考（出门时别忘了带，尤其是如果你是系统管理员）。售价 4.95 美元，但遗憾的是不再提供免费下载。

*

Arnold Robbins，*有效的 Awk 编程*，自由软件基金会 / O'Reilly and Associates，2000，1-882114-26-4。

绝对的最好的 awk 教程和参考。这本书的免费电子版是*awk*文档的一部分，最新版本的印刷版可以从 O'Reilly and Associates 获得。

这本书为*ABS 指南*的作者提供了灵感。

*

Bill Rosenblatt，*学习 Korn Shell*，O'Reilly and Associates，1993，1-56592-054-6。

这本写得很好的书包含了一些关于 shell 脚本的一般性优秀建议。

*

Paul Sheer，*LINUX：Rute 用户教程和展示*，第 1 版，，2002，0-13-033351-4。

对 Linux 系统管理的非常详细和易读的介绍。

这本书有印刷版，或[在线](http://burks.brighton.ac.uk/burks/linux/rute/rute.htm)版。

*

Ellen Siever 和 O'Reilly and Associates 的员工，*Linux 精华*，第 2 版，O'Reilly and Associates，1999，1-56592-585-8。

综合来说最好的 Linux 命令参考。它甚至有一个 Bash 部分。

*

Dave Taylor，*Wicked Cool Shell Scripts: 101 Scripts for Linux, Mac OS X, and Unix Systems*，第 1 版，No Starch Press，2004，1-59327-012-7。

几乎正如标题所承诺的那样 . . .

*

*《UNIX CD 书架》*，第 3 版，O'Reilly and Associates，2003，0-596-00392-7。

一套包含七本 UNIX 书籍的光盘，包括*UNIX Power Tools*、*Sed and Awk*和*Learning the Korn Shell*。所有您可能需要的 UNIX 参考和教程的完整集合，大约 130 美元。即使这意味着要负债和拖欠房租，也要买这套书。

更新：似乎已经绝版了。唉。您仍然可以购买这些书的实体版。

*

O'Reilly 出版的 Perl 书籍。（实际上，*任何* O'Reilly 的书籍。）

**其他资源**

Fioretti, Marco, "Scripting for X Productivity," *Linux Journal*，第 113 期，2003 年 9 月，第 86-9 页。

Ben Okopnik 在[*Linux Gazette*](http://www.linuxgazette.net)第 53、54、55、57 和 59 期上发表的精心撰写的*Bash 脚本入门*文章，以及他在第 56 期对“Bash 的深层、黑暗秘密”的解释。

Chet Ramey 的*Bash - The GNU Shell*，这是在 1994 年 7 月-8 月号的[*Linux Journal*](http://www.linuxjournal.com)上发表的两篇系列文章。

Mike G 的[Bash-Programming-Intro HOWTO](http://www.tldp.org/HOWTO/Bash-Prog-Intro-HOWTO.html)。

Richard 的[Unix 脚本宇宙](http://www.injunea.demon.co.uk/index.htm)。

Chet Ramey 的[Bash FAQ](http://tiswww.case.edu/php/chet/bash/FAQ)。

[Greg 的 WIKI: Bash FAQ](http://wooledge.org:8000/BashFAQ)。

在[Lucc 的 Shell 脚本](http://alge.anart.no/linux/scripts/)上可以找到示例 shell 脚本。

在[SHELLdorado](http://www.shelldorado.com)上可以找到示例 shell 脚本。

在[Noah Friedman 的脚本网站](http://www.splode.com/~friedman/software/scripts/src/)上可以找到示例 shell 脚本。

来自 Albing, Vossen 和 Newham 所著的*《Bash 脚本编程秘籍》*的[示例](http://bashcookbook.com/bashinfo/)。

在[zazzybob](http://www.zazzybob.com)上可以找到示例 shell 脚本。

Steve Parker 的[Shell Programming Stuff](http://steve-parker.org/sh/sh.shtml)。实际上，他所有的 shell 脚本书籍都强烈推荐。还可以参考 Steve 的[用 shell 脚本编写的 Arcade 游戏](http://nixshell.wordpress.com/2011/07/13/arcade-games-written-in-a-shell-script/)。

在[Bash Hackers Wiki](http://www.bash-hackers.org/wiki.doku.php)上有一系列优秀的 Bash 脚本技巧、窍门和资源。

Giles Orr 的[Bash-Prompt HOWTO](http://www.tldp.org/HOWTO/Bash-Prompt-HOWTO/)。

[Pixelbeat 命令行参考](http://www.pixelbeat.org/cmdline.html)。

非常好的**sed**、**awk**和正则表达式教程，可在[The UNIX Grymoire](http://www.grymoire.com/Unix/index.html)找到。

GNU 的[sed](http://www.gnu.org/software/sed/manual/)和[gawk](http://www.gnu.org/software/gawk/manual/)手册。如您所忆，gawk 是增强版的 GNU **awk**。

在[seder 的抓袋](http://sed.sourceforge.net/grabbag/)中有许多有趣的 sed 脚本。

在[Linux Reviews](http://linuxreviews.org)中有技巧和窍门。

Trent Fisher 的[groff 教程](http://www.cs.pdx.edu/~trent/gnu/groff/groff.html)。

David Wheeler 的[Shell 中的文件名](http://www.dwheeler.com/essays/filenames-in-shell.html)论文。

[Shelltris]和[shellitaire]在[Shell Script Games](http://www.shellscriptgames.com)。

YongYe 的复杂而精彩的[Tetris 游戏脚本](http://bash.deta.in/Tetris_Game.sh)。

Mark Komarinski 的[打印使用指南](http://www.tldp.org/HOWTO/Printing-Usage-HOWTO.html)。

[Linux USB 子系统](http://www.linux-usb.org/USB-guide/book1.html)（有助于编写影响 USB 外设的脚本）。

在[University of Alberta site](http://sunsite.ualberta.ca/Documentation)的[文本工具文档的第十章](http://sunsite.ualberta.ca/Documentation/Gnu/textutils-2.0/html_chapter/textutils_10.html)中，有关于输入/输出重定向的一些很好的材料。

Rick Hohensee 编写了完全用 Bash 脚本编写的*i386*汇编器*osimpa*。

*dgatwood*有一个非常棒的[shell 脚本游戏](http://www.shellscriptgames.com/)网站，特色是一个 Tetris 克隆和单人纸牌游戏。

Aurelio Marinho Jargas 编写了一个[正则表达式向导](http://txt2regex.sf.net)。他还编写了一本关于正则表达式的[葡萄牙语书籍](http://guia-er.sf.net)。

Ben Tomkins 创建了[Bash Navigator](http://bashnavigator.sourceforge.net)目录管理工具。

William Park 一直在努力将某些*Awk*和*Python*功能集成到 Bash 中。其中之一是*gdbm*接口。他在[Freshmeat.net](http://freshmeat.net)上发布了*bashdiff*。他在 2004 年 11 月的[*Linux Gazette*](http://www.linuxgazette.net)杂志上发表了一篇关于向 Bash 添加字符串函数的[文章](http://linuxgazette.net/108/park.html)，在 12 月号的[后续文章](http://linuxgazette.net/109/park.html)中，以及在 2005 年 1 月的[另一篇文章](http://linuxgazette.net/110/park.htm)中。

Peter Knowles 编写了一个[复杂的 Bash 脚本](http://booklistgensh.peterknowles.com/)，该脚本可以在[索尼 Librie](http://www.dottocomu.com/b/archives/002571.html)电子书阅读器上生成书单。这个有用的工具简化了在*Librie*（以及更新的*PRS-xxx 系列*设备）上加载非 DRM 用户内容的过程。

Tim Waugh 的[xmlto](http://cyberelk.net/tim/xmlto/)是一个用于将 Docbook XML 文档转换为其他格式的复杂 Bash 脚本。

Philip Patterson 的[logforbash](http://www.gossiplabs.org)日志/调试脚本。

[AuctionGallery](http://auctiongallery.sourceforge.net)，一个用 Bash 编写的针对 eBay "power sellers"的应用程序。

历史上有兴趣的是 Colin Needham 的 *原始国际电影数据库 (IMDB) 读者投票脚本*，这些脚本很好地说明了使用 awk 进行字符串解析的方法。不幸的是，URL 链接已损坏。

---

Fritz Mehner 为 *vim* 文本编辑器编写了一个 [bash-support 插件](http://vim.sourceforge.net/scripts/script.php?script_id=365)。他还编写了自己的 [Bash 样式表](http://lug.fh-swf.de/vim/vim-bash/StyleGuideShell.en.pdf)。与 ABS Guide Unofficial Stylesheet 进行比较。

---

*Penguin Pete* 在他的 [卓越网站](http://www.penguinpetes.com) 上有相当多的 shell 脚本技巧和提示。强烈推荐。

由 Chet Ramey 和 Brian Fox 编写的优秀的 *Bash 参考手册*，作为 *bash-2-doc* 软件包的一部分分发（作为 rpm 提供）。请特别注意此软件包中的说明性示例脚本。

John Lion 的经典之作，[*《第六版 UNIX 操作系统评注》*](http://www.lemis.com/grog/Documentation/Lions/index.html)。

comp.os.unix.shell 新闻组。

[Linux Questions](http://www.linuxquestions.org) 上的 [*dd* 线程](http://www.linuxquestions.org/questions/showthread.php?t=362506)。

[comp.os.unix.shell FAQ](http://www.newsville.com/cgi-bin/getfaq?file=comp.unix.shell/comp.unix.shell_FAQ_-_Answers_to_Frequently_Asked_Questions)。

各种 comp.os.unix [FAQs](http://www.faqs.org/faqs/by-newsgroup/comp/comp.unix.shell.html)。

覆盖 dc 的 [*Wikipedia* 文章](http://en.wikipedia.org/wiki/Dc_(Unix))。

**bash** 和 **bash2**、**date**、**expect**、**expr**、**find**、**grep**、**gzip**、**ln**、**patch**、**tar**、**tr**、**bc**、**xargs** 的 manpages。关于 **bash**、**dd**、**m4**、**gawk** 和 **sed** 的 *texinfo* 文档。
