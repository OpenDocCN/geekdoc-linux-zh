# 附录 I. 重要系统目录

> 原文：[`tldp.org/LDP/abs/html/systemdirs.html`](https://tldp.org/LDP/abs/html/systemdirs.html)

系统管理员和任何编写管理脚本的都应该熟悉以下系统目录。

+   `/bin`

    二进制文件（可执行文件）。基本系统程序和实用工具（例如 **bash**）。

+   `/usr/bin` [[1]](#FTN.AEN23969)

    更多系统二进制文件。

+   `/usr/local/bin`

    特定机器的杂项二进制文件。

+   `/sbin`

    系统二进制文件。基本系统管理程序和实用工具（例如 **fsck**）。

+   `/usr/sbin`

    更多系统管理程序和实用工具。

+   `/etc`

    *等等*. 系统级配置脚本。

    特别值得注意的是文件`/etc/fstab`（文件系统表）、`/etc/mtab`（已挂载的文件系统表）和文件`/etc/inittab`。

+   `/etc/rc.d`

    在 Red Hat 和 Linux 的衍生发行版中的启动脚本。

+   `/usr/share/doc`

    已安装软件包的文档。

+   `/usr/man`

    系统级的手册页。

+   `/dev`

    设备目录。物理和虚拟设备的条目（但不是挂载点）。见第二十九章。

+   `/proc`

    进程目录。包含有关运行进程和内核参数的信息和统计。见第二十九章。

+   `/sys`

    系统级设备目录。包含有关设备和设备名称的信息和统计。这是 Linux 2.6.X 内核中新增的。

+   `/mnt`

    *挂载*. 用于挂载硬盘分区，例如 `/mnt/dos`，以及物理设备。在新版的 Linux 发行版中，`/media` 目录已取代成为 I/O 设备的首选挂载点。

+   `/media`

    在新版的 Linux 发行版中，I/O 设备（如 CD/DVD 驱动器或 USB 闪存驱动器）的首选挂载点。

+   `/var`

    *可变*（可更改）的系统文件。这是一个 Linux/UNIX 机器运行时生成数据的“便笺”目录。

+   `/var/log`

    系统级日志文件。

+   `/var/spool/mail`

    用户邮件队列。

+   `/lib`

    系统级库文件。

+   `/usr/lib`

    更多系统级库文件。

+   `/tmp`

    系统临时文件。

+   `/boot`

    系统启动目录。内核、模块链接、系统映射和启动管理器都存放在这里。

### 备注

| [[1]](systemdirs.html#AEN23969) | 一些早期的 UNIX 系统有一个快速、容量小的固定硬盘（包含`/`，根分区），以及一个更大的、但较慢的第二个硬盘（包含`/usr`和其他分区）。因此，最常用的程序和实用工具因此驻留在小而快的硬盘上，在`/bin`中，而其他则驻留在较慢的硬盘上，在`/usr/bin`中。这也解释了`/sbin`和`/usr/sbin`、`/lib`和`/usr/lib`等之间的分割。 |
