# 附录 Q. 下载和镜像站点

> 原文：[`tldp.org/LDP/abs/html/mirrorsites.html`](https://tldp.org/LDP/abs/html/mirrorsites.html)

此文档的最新更新，作为一个包含 SGML 源代码和渲染 HTML 的归档 bzip2-ed "tarball"，可以从[作者的网站](http://bash.deta.in/abs-guide-latest.tar.bz2)下载。还有一个[PDF 版本](http://bash.deta.in/abs-guide.pdf)，[镜像站点](http://www.mediafire.com/file/xi34ape1bifcnlb/abs-guide.pdf)。同样还有一个[epub 版本](http://bash.deta.in/abs-guide.epub)，由 Craig Barnes 和 Michael Satke 提供。[变更日志](http://bash.deta.in/Change.log)提供了详细的修订历史。*ABS 指南*甚至有自己的[freecode 页面](http://freecode.com/projects/advancedbashscriptingguide/)，用于跟踪项目的主要更新、用户评论和流行度评分。

此文档的遗产托管站点是[Linux 文档项目](http://www.tldp.org/LDP/abs/)，该站点维护了许多其他指南和 HOWTO。

非常感谢 Ronny Bangsund 捐赠[服务器空间](http://bash.deta.in/)来托管此项目。
