# 38.5. 致谢

> 原文：[`tldp.org/LDP/abs/html/credits.html`](https://tldp.org/LDP/abs/html/credits.html)

*社区参与使这个项目成为可能。* 作者衷心感谢所有这些人的帮助和反馈，没有他们的帮助，撰写这本书是不可想象的。

Philippe Martin 将这份文档的第一版（0.1）翻译成了 DocBook/SGML。尽管他不是在一家小型法国公司作为软件开发人员工作，但他喜欢从事 GNU/Linux 文档和软件的工作，阅读文学作品，演奏音乐，并且为了他的平静心情，与朋友们一起欢乐。你可能在法国或巴斯克地区找到他，或者你可以通过 feloy@free.fr 发送电子邮件给他。

Philippe Martin 也指出，可以使用大括号表示法使用超过 $9 的位置参数。（参见示例 4-5）。

Stphane Chazelas 发送了一份长长的修正、补充和示例脚本的列表。他不仅仅是一个贡献者，实际上，在一段时间里，他承担了这份文档的**共同编辑**角色。**非常感谢！**

Paulo Marcel Coelho Aragao 提供了许多修正，无论是主要的还是次要的，并贡献了许多有用的建议。

我特别想感谢 *Patrick Callahan*、*Mike Novak* 和 *Pal Domokos*，他们发现了错误，指出了歧义，并在文档的初步版本（0.1）中提出了澄清和建议和更改。他们对 shell 脚本和一般文档问题的热烈讨论激发了我尝试使这份文档更易于阅读。

我感谢 Jim Van Zandt 指出这份文档 0.2 版本中的错误和遗漏。他还贡献了一个指导性的示例脚本。

感谢 Jordi Sanfeliu 允许使用他的优秀树形脚本（示例 A-16），以及 Rick Boivie 对其进行的修订。

同样，感谢 Michel Charpentier 允许使用他的 dc 分解脚本（示例 16-52）。

感谢 Noah Friedman 允许使用他的字符串函数脚本（示例 A-18）。

Emmanuel Rouat 在命令替换、别名和路径管理方面提出了修正和建议。他还贡献了一个非常棒的 `.bashrc` 示例文件（附录 M）。

Heiner Steven 仁慈地允许使用他的基数转换脚本，示例 16-48。他还做了一些修正和许多有用的建议。特别感谢。

Rick Boivie 贡献了令人愉快的递归 *pb.sh* 脚本（示例 36-11），修订了 *tree.sh* 脚本（示例 A-16），并为 *monthlypmt.sh* 脚本（示例 16-47）提出了性能改进建议。

Florian Wisser 在测试字符串的一些细微之处（见 示例 7-6）和其他事项上启发了我。

Oleg Philon 提出了关于 cut 和 pidof 的建议。

Michael Zick 将 空数组 示例扩展，以展示一些令人惊讶的数组属性。他还贡献了 *isspammer* 脚本（示例 16-41 和 示例 A-28）。

Marc-Jano Knopp 提供了有关 DOS 批处理文件的纠正和澄清。

Hyun Jin Cha 在进行韩语翻译的过程中发现了文档中的几个错误。感谢指出这些错误。

Andreas Abraham 发送了一份长长的错别字和其他纠正列表。特别感谢！
