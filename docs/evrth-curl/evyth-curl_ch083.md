# 下载

“下载”意味着从网络上的服务器获取数据，此时服务器显然被认为是“高于”你的。这是将数据从服务器加载到运行 curl 的你的机器上。

下载可能是 curl 最常见的使用场景——将 URL 指向的特定数据检索到你的机器上。
