### **Crackmapexec**简介

Crackmapexec 是一个开发后工具，用于自动评估大型活动目录网络的安全性。Crackmapexec 利用内部活动目录协议来实现其功能，使其能够规避大多数端点保护/IDS/IPS 解决方案。需要注意的是，Crackmapexec 使用的是 Impacket 库和 PowerSploit 工具包。

