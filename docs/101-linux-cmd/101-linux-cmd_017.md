# `who` 命令

`who` 命令让您打印出登录用户列表、系统的当前运行级别以及上次系统启动时间。

### 示例

1.  打印当前登录用户的全部详细信息

```sh
      who -a 

```

1.  打印所有已死进程的列表

```sh
      who -d -H 

```

### 语法：

```sh
      who [options] [filename] 

```

### 其他标志及其功能

| **短标志** | **描述** |
| --- | --- |
| `-r` | 打印所有当前运行级别 |
| `-d` | 打印所有已死进程 |
| `-q` | 打印所有登录名和总登录用户数 |
| `-h` | 打印显示的列标题 |
| `-b` | 打印上次系统启动时间 |
