# `nice/renice` 命令

`nice/renice` 命令用于修改将要执行的程序的优先级。优先级范围在 -20 到 19 之间，其中 19 是最低优先级。

### 示例：

1.  在后台以低于默认优先级（较慢）运行 cc 命令：

```sh
      nice -n 15 cc -c *.c & 

```

1.  将优先级提升到属于“test”组的所有进程：

```sh
      renice --20 -g test 

```

### 语法：

```sh
      nice [  -Increment|  -n Increment ] Command [ Argument ... ] 

```

### 标志：

| **短标志** | **长标志** | **描述** |
| :-- | :-- | :-- |
| `-增量` | - | 增量是你想要分配的优先级值。 |
| `-n 增量` | - | 与 `-增量` 相同 |
