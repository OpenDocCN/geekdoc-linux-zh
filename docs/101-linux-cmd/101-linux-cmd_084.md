# `du` 命令

`du` 命令，即 `disk usage` 的缩写，允许你检索指定目录中磁盘空间使用情况的信息。为了根据你需要的信息自定义输出，这个命令可以与适当的选项或标志搭配使用。

### 示例：

1.  要显示当前目录中子目录的估计大小：

```sh
      du 

```

1.  要显示指定目录内子目录的估计大小：

```sh
      du {PATH_TO_DIRECTORY} 

```

### 语法：

```sh
      du [OPTION]... [FILE]...
du [OPTION]... --files0-from=F 

```

### 其他标志及其功能：

*注意：此列表并非选项的完整列表。*

| **短标志** | **长标志** | **描述** |
| :-- | :-- | :-- |
| `-a` | `--all` | 包括文件和目录的信息 |
| `-c` | `--total` | 在文件/目录列表的末尾提供总计数 |
| `-d` | `--max-depth=N` | 提供从执行命令的目录开始的最多 `N` 级别的信息 |
| `-h` | `--human-readable` | 以人类可读的单位显示文件大小，而不是字节 |
| `-s` | `--summarize` | 仅显示总文件大小，而不是文件/目录列表 |
