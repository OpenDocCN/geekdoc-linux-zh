# `man` 命令

`man` 命令用于显示我们可以在终端上运行的任何命令的手册。它提供的信息包括：描述、选项、作者等。

### 示例：

1.  `printf` 的手册页

```sh
      man printf 

```

1.  介绍部分的第二部分手册页：

```sh
      man 2 intro 

```

1.  使用 `-l` 标志查看本地文件的手册：

```sh
      man -l [LOCAL-FILE] 

```

### 语法：

```sh
      man [SECTION-NUM] [COMMAND NAME] 

```

### 额外的标志及其功能：

| **短标志** | **长标志** | **描述** |
| :-- | :-- | :-- |
| `-f` | - | 返回命令的章节 |
| `-a` | - | 显示所有命令的手册页 |
| `-k` | - | 在所有手册页中使用正则表达式搜索给定命令 |
| `-w` | - | 返回给定命令手册页的位置 |
| `-I` | - | 按大小写敏感搜索命令手册 |
