# `last`命令

此命令会显示自`var/log/wtmp`文件创建以来所有登录和登出的用户列表。你还可以添加一些参数，例如显示特定用户何时登录以及登录了多久。

如果你想要查看最后 5 条日志，只需在命令中添加`-5`，如下所示：

```sh
      last -5 

```

如果你想要查看最后 10 条记录，请添加`-10`。

另一个很酷的功能是，如果你添加`-F`，你可以看到包括日期在内的登录和登出时间。

```sh
      last -F 

```

使用此命令可以查看很多东西。如果你需要了解更多关于此命令的信息，你可以运行：

```sh
      last --help 

```
