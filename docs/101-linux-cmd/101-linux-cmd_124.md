# `quota` 命令

`quota` 命令用于显示磁盘使用情况和限制。

### 安装：

您可以直接运行以下命令在 ubuntu 系统上安装 quota：

```sh
      sudo apt-get install quota 

```

对于 Debian 系统，您可以使用不带 sudo 的安装命令：

```sh
      apt-get install quota 

```

### 语法：

`quota` 命令的一般语法如下：

```sh
      quota [ -u [ User ] ] [ -g [ Group ] ] [ -v | -q ] 

```
