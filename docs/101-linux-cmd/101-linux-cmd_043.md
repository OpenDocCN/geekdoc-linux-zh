# `clear` 命令

在 Linux 中，`clear`命令用于清除终端屏幕。

## 示例

```sh
      $ clear 

```

## 之前：

```sh
      $ echo Hello World
Hello World

$ clear 

```

## 执行`clear`命令后：

```sh
      $ 

```
