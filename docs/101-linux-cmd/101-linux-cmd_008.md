# `cal` 命令

`cal` 命令在终端中显示格式化的日历。如果没有指定选项，`cal` 将显示当前月份，并突出显示当前日期。

### 语法：

```sh
      cal [general options] [-jy] [[month] year] 

```

### 选项：

| **选项** | **描述** |
| :-- | :-- |
| `-h` | 不突出显示今天的日期。 |
| `-m month` | 指定要显示的月份。月份指定可以是完整的月份名称（例如，二月），至少三个字母的月份缩写（例如，Feb），或数字（例如，2）。如果您指定一个数字，后跟字母“f”或“p”，则分别显示下一年或前一年的该月。例如，`-m 2f` 将显示下年的二月。 |
| `-y year` | 指定要显示的年份。例如，`-y 1970` 将显示 1970 年整年的日历。 |
| `-3` | 显示上个月、这个月和下个月。 |
| `-1` | 仅显示这个月。这是默认选项。 |
| `-A num` | 显示在已指定月份之后发生的 num 个月。例如，`-3 -A 3` 将显示上个月、这个月和之后的四个月；以及`-y 1970 -A 2` 将显示 1970 年中的每个月，以及 1971 年的前两个月。 |
| `-B num` | 显示在已指定月份之前发生的 num 个月。例如，`-3 -B 2` 将显示前三个月、这个月和下个月。 |
| `-d YYYY-MM` | 假设当前月份是 YYYY 年 MM 月。 |

### 示例：

1.  显示这个月的日历，并突出显示今天。

```sh
      cal 

```

1.  与上一个命令相同，但不突出显示今天。

```sh
      cal -h 

```

1.  显示上个月、这个月和下个月。

```sh
      cal -3 

```

1.  显示整个年份的日历。

```sh
      cal -y 

```

1.  显示 2000 年整年的日历。

```sh
      cal -y 2000 

```

1.  与上一个命令相同。

```sh
      cal 2000 

```

1.  显示今年 12 月的日历。

```sh
      cal -m [December, Dec, or 12] 

```

1.  显示 2000 年 12 月的日历。

```sh
      cal 12 2000 

```
