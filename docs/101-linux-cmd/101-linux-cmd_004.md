# `head` 命令

`head` 命令打印文件的第一个十行。

示例：

```sh
      head filename.txt 

```

语法：

```sh
      head [OPTION] [FILENAME] 

```

### 获取特定数量的行：

使用 `-n` 选项和一个数字（应为整数）来显示行数。

示例：

```sh
      head -n 10 foo.txt 

```

此命令将显示文件 `foo.txt` 的前十行。

语法：

```sh
      head -n <number> foo.txt 

```

### 其他标志及其功能

| **短标志** | **长标志** | **描述** |
| :-- | :-- | :-- |

| `-c` | `--bytes=[-]NUM` | 打印每个文件的第一个 NUM 字节；带有前导 '-'，

打印每个文件的除最后一行外的所有字节 |

| `-n` | `--lines=[-]NUM` | 打印前 NUM 行，而不是前 10 行；带有前导 '-'，

打印除最后一行外的每一行的所有内容 |

| `-q` | `--quiet, --silent` | 从不打印包含文件名的标题 |
| --- | --- | --- |
| `-v` | `--verbose` | 总是打印包含文件名的标题 |
| `-z` | `--zero-terminated` | 行分隔符是 NUL，而不是换行符 |
|  | `--help` | 显示此帮助信息并退出 |
|  | `--version` | 输出版本信息并退出 |
