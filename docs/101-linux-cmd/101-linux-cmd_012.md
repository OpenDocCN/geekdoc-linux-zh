# `factor` 命令

`factor` 命令打印每个指定的整数 `NUMBER` 的质因数。如果命令行上没有指定，它将从标准输入读取。

## 语法

```sh
      $ factor [NUMBER]... 

```

OR:

```sh
      $ factor OPTION 

```

## 选项

| **选项** | **描述** |
| :-- | :-- |
| `--help` | 显示此帮助信息并退出。 |
| `--version` | 输出版本信息并退出。 |

## 示例

1.  打印质数的质因数。

```sh
      $ factor 50 

```

1.  打印非质数的质因数。

```sh
      $ factor 75 

```
