# 查看 Linux 上的当前用户

> 原文：[`techbyexample.com/current-user-linux/`](https://techbyexample.com/current-user-linux/)

# **概述**

Linux 命令 **‘whoami’** 可以用来查看当前登录的用户。

让我们来看一下这个命令的实际操作。在终端中输入命令

```go
whoami
```

它将打印当前用户。假设当前用户是**john**，那么运行上述命令将仅打印**john**

**输出**

```go
whoami
john
```
